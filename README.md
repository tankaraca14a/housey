# Housey - Dalmatian Coast Vacation Rental

A beautiful website for a house rental on the Dalmatian coast.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- Next.js 15
- React 18
- TypeScript
- Tailwind CSS
- React Hook Form + Zod
